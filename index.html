Aprenda Ionic 2
com um curso 100% mão na massa

Aprenda na prática como criar aplicações para iOS, Android e Windows Phone
utilizando um dos melhores frameworks mobile do mercado, juntamente com o Angular 2.

Objetivo

Vamos aprender a criar aplicações utilizando o Ionic 2, ver as principais diferenças entre a versão 1 e 2, vamos conhecer os novos recursos do Ionic, entender como funciona o novo framework e seu ambiente, que utiliza o TypeScript (como dependência do Angular 2). Vamos criar 2 aplicações durante nossa aula, utilizando os recursos de Mapa, GPS, Notificações, Cards, Forms Toasts e muito mais...

Ao final do curso vamos fazer a publicação de uma das aplicações para a PlayStore(do Google) e ver como é a rotina de publicação para a AppStore.
